# 簡介
一款基於 Pygame 模組開發的 2D 道路躲避遊戲，玩家需控制車輛，左右移動以躲避來自前方的車輛。遊戲包括主選單、控制說明頁面、倒數計時、暫停和重啟等功能。隨著等級提升，車速會逐步加快(也可自行加減速)，提升遊戲難度。遊戲結束時顯示 "GAME OVER" 並允許玩家重新開始或退出
# 安裝 pygame
```
python -m pip install -U pygame --user
```
